# DSA Learning App

A responsive web application for learning Data Structures and Algorithms, optimized for both desktop and mobile (iOS) browsers.

## Features

- Responsive design that works on desktop and mobile
- Mobile-first approach with touch-friendly interfaces
- Dark mode support
- Optimized for iOS Safari

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React 19

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

For mobile testing, use your computer's IP address (e.g., http://192.168.x.x:3000) on your iPhone when both devices are on the same network.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components
- `lib/` - Utility functions and helpers

## Build for Production

```bash
npm run build
npm start
```
